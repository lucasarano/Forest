# Forest

AI-powered learning platform with an interactive Learning Tree.

## Environment variables

Copy `.env.example` to `.env` and set:

| Variable | Description |
|----------|-------------|
| `VITE_SUPABASE_URL` | Your Supabase project URL (auth) |
| `VITE_SUPABASE_ANON_KEY` | Your Supabase anonymous key (auth) |
| `VITE_OPENAI_API_KEY` | Your OpenAI API key (Learning Tree AI) |

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Open http://localhost:5173 (or the URL shown in the terminal).
